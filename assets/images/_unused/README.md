# Unused image assets

These files were part of the original [vCard Personal Portfolio](https://github.com/codewithsadee/vcard-personal-portfolio) template by **codewithsadee** (MIT‑licensed), or were intermediate working files (architecture diagrams, alternative logos, etc.).

They are **not referenced from `index.html`** and are kept here for reference only — so the active `assets/images/` directory contains only assets that are actually shipped.

If a file in this folder becomes useful again, move it back into `assets/images/` (or the appropriate sub‑folder) and reference it from `index.html`.

Safe to delete this whole `_unused/` folder if you no longer need any of these files.
